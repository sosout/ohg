# ohg ![download](https://img.shields.io/npm/dt/ohg.svg) ![npm-version](https://img.shields.io/npm/v/ohg.svg) ![license](https://img.shields.io/npm/l/ohg.svg)

> 十分简单的Git CLI脚手架.

### 安装
ohg 依赖 [Node.js](https://nodejs.org/en/) (>=6.x)：

```
$ npm install ohg -g
```

### 用法
```
$ ohg create
```
